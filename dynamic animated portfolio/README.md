# Nilay Bhattacharya - Developer Portfolio

A modern, fully responsive dark-themed portfolio website built with HTML, CSS, and JavaScript. Features include animated sections, glassmorphism UI, smooth scrolling, and interactive elements.

## 📋 Project Structure

```
portfolio/
├── index.html           # Main HTML file
├── style.css            # Styling and animations
├── script.js            # Interactive features
├── README.md            # This file
└── assets/
    ├── logo.jpg         # Your profile image (NEEDS TO BE ADDED)
    └── favicon.svg      # Website favicon
```

## 🚀 Quick Start

1. **Clone or extract the project** to your local machine
2. **Add your profile image**: 
   - Save your profile photo as `logo.jpg`
   - Place it in the `assets/` folder
   - Alternatively, replace `assets/logo.jpg` references with your image URL

3. **Open in browser**: Simply open `index.html` in your web browser

## 🎨 Customization

### Update Personal Information
Edit the following in `index.html`:
- Your name and title
- Contact information (phone, email)
- Social links (GitHub, LinkedIn)
- Project descriptions
- Skills and experience

### Update Styling
Modify variables in `assets/css/style.css`:
```css
:root {
  --bg: #070a12;              /* Background color */
  --primary: #5d61ff;         /* Primary accent color */
  --accent: #00e5ff;          /* Secondary accent color */
  --text: #e6e8ff;            /* Text color */
  /* ... more variables */
}
```

### Update JavaScript Behavior
Edit `assets/js/script.js` to:
- Change typing animation phrases
- Modify animation timings
- Customize form handling
- Update navigation behavior

## 📸 Adding Your Logo

To use your profile image:

1. **Save your image as `logo.jpg`** in the `assets/` folder
2. Image will automatically appear in:
   - Navigation bar (small circular icon)
   - Hero section (main profile photo)
   - About section (beside introduction text)

**Recommended image specs:**
- Format: JPG, PNG, or WebP
- Aspect ratio: Square (1:1)
- Minimum size: 400x400 pixels
- File size: Under 200KB for optimal performance

## 🚀 Deployment

### GitHub Pages
1. Push your files to a GitHub repository
2. Go to Settings → Pages
3. Select main branch as source
4. Your site will be live at `username.github.io/portfolio`

### Netlify
1. Connect your GitHub repo
2. Set build command: (leave empty)
3. Set publish directory: `.`
4. Deploy

### Vercel
1. Import your GitHub repository
2. Click Deploy
3. Your site is live
