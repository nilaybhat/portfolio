## 🆘 Troubleshooting

### Logo not showing?
→ Make sure file is named exactly `logo.jpg` (lowercase)
→ Make sure it's in `assets/` folder
→ Supported formats: JPG, PNG, WebP

### Changes not updating?
→ Hard refresh browser: `Ctrl + Shift + R`
→ Clear cache: `Ctrl + Shift + Del`
→ Use different browser to test

### Mobile menu not working?
→ Use DevTools (F12) to test responsive
→ Check browser console for errors

### Form not sending emails?
→ Contact form is demo only
→ See CUSTOMIZE.md to add real email backend

### What to do if...

**I want to add a new section**
